##Descripción

Este proyecto implementa una **Simulación de Eventos Discretos (DES)** utilizando SimPy para modelar la ejecución de procesos en un sistema operativo de tiempo compartido.

Los procesos:

- Solicitan memoria RAM (Container).
- Esperan CPU (Resource).
- Ejecutan instrucciones.
- Pueden entrar en estado de espera (I/O).
- Finalizan y liberan recursos.

Se analizaron distintos niveles de carga y estrategias de optimización.
